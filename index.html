Code Learner

This was a weekend project after our week of machine learning. I pair programmed this assignment with Daniel Newell. We were assigned to create a classifier that would be able to tell the difference between 10 different programming languages. To give the learner more data we created a web scraper to gather code examples from Rosetta Code. With our code examples, we created a classifier that could correctly guess the code type from snippets of code over 90% of the time.
